Dino8 is a retro game engine written in C with Lua scripting. Write games with a small, focused API for drawing pixels, handling input, playing audio, and managing tilemaps — then package and share them on the web or desktop.

No token limits, no restricted memory, no forced resolution. Use as many sprites, sounds, and lines of code as you want.

FEATURES

Lua Scripting

Write game logic in Lua 5.4 with hot reload during development.

16-Color Palette

Dawnbringer 16 default palette with custom palette support.

Pixel Graphics

Sprites, shapes, tilemaps, camera, clipping, and render targets.

Audio

Music and sound effects with named clips, pitch, and volume control.

Web Export

Bundle games into a single HTML file. Push to itch.io with one command.

Packaging

Create .fos packages for distribution. Validate and inspect projects.

GETTING STARTED

# create a new game
dino8 create my-game

# run it
dino8 my-game/

API SAMPLE

function _init()
  x, y = 64, 64
end

function _update()
  if btn(0) then x = x - 1 end
  if btn(1) then x = x + 1 end
  if btn(2) then y = y - 1 end
  if btn(3) then y = y + 1 end
end

function _draw()
  cls()
  circ(x, y, 4, 7)
  print("hello dino8", 40, 10, 14)
end

CLI

dino8 my-game/              # run a project
dino8 my-game.fos           # run a package
dino8 create my-game        # scaffold a new game
dino8 fossilize my-game     # create .fos package
dino8 validate my-game/     # check project structure
dino8 export web my-game/   # single-file HTML export
dino8 push my-game/         # publish to itch.io
dino8 help                  # all commands
