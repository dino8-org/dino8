Dino8 is a 2d retro game toolkit for developers who want PICO-8’s simplicity but without the limits. Write Lua, use your favorite tools, and ship to web, desktop, and mobile from one CLI.

No token limits, no restricted memory, no forced resolution. Bring your own tools and focus on making the game.

FEATURES

Lua Scripting

Write game logic in Lua 5.4 with hot reload during development.

16-Color Palette

Dawnbringer 16 default palette with custom palette support.

Pixel Graphics

Sprites, shapes, tilemaps, camera, clipping, and render targets.

Audio

Music and sound effects with named clips, pitch, and volume control.

Web Export

Bundle games into a single HTML file. Push to itch.io with one command.

Packaging

Create .fos packages for distribution. Validate and inspect projects.

GETTING STARTED

# create a new game
dino8 create my-game

# run it
dino8 my-game/

API SAMPLE

function _init()
  x, y = 64, 64
end

function _update()
  if btn(0) then x = x - 1 end
  if btn(1) then x = x + 1 end
  if btn(2) then y = y - 1 end
  if btn(3) then y = y + 1 end
end

function _draw()
  cls()
  circ(x, y, 4, 7)
  print("hello dino8", 40, 10, 14)
end

CLI

dino8 my-game/              # run a project
dino8 my-game.fos           # run a package
dino8 create my-game        # scaffold a new game
dino8 fossilize my-game     # create .fos package
dino8 validate my-game/     # check project structure
dino8 export web my-game/   # single-file HTML export
dino8 export ios my-game/   # export Xcode project
dino8 push my-game/         # publish to itch.io
dino8 examples              # download example projects
dino8 help                  # all commands
dino8 my-game/

Run a game from a project directory. Auto-detects conf.lua and launches your game with hot reload — edit your code and see changes instantly without restarting.

dino8 my-game.fos

Run a packaged .fos file. Fossils are self-contained game archives — share them with players or use them for distribution.

dino8 create my-game

Scaffold a new game project with the standard directory structure, a starter conf.lua, and a minimal main.d8 entry point. The fastest way to start building.

dino8 fossilize my-game

Package your project into a single .fos archive. Bundles all code, assets, and config into a portable file that can be run, shared, or uploaded.

dino8 validate my-game/

Check your project structure before packaging. Verifies conf.lua is valid, referenced assets exist, and the entry point is reachable — catches mistakes early.

dino8 export web my-game/

Export your game as a single self-contained HTML file. Drop it on any web server or upload to itch.io — no server-side code needed.

dino8 export ios my-game/

Export your game as an Xcode project ready for iOS. Generates the full project structure so you can build and deploy to iPhone and iPad.

dino8 push my-game/

Publish your game directly to itch.io using butler. Handles fossilizing, web export, and upload in one step — configure your itch table in conf.lua and go.

dino8 info my-game.fos

Display metadata about a .fos file or project — title, author, version, file count, and size. Useful for inspecting packages before distributing.

dino8 list my-game.fos

Show the file tree inside a .fos archive. Handy for verifying what got packaged or debugging missing assets.

dino8 extract my-game.fos

Extract a .fos archive back into a directory. Useful for inspecting packaged games, modding, or recovering source from a distributed fossil.

dino8 test my-game/

Run unit tests for your game in headless mode. Executes your test suite and reports results without opening a window.

dino8 simulate my-game/

Run your game in headless simulation mode — no window, no audio. Great for automated testing, CI pipelines, or verifying game logic without a display.

dino8 examples

Download the official example projects. A quick way to explore what dino8 can do and learn from working code.

dino8 update

Update the dino8 engine to the latest version. Checks for a newer release and installs it in place.

dino8 help

Show the full list of commands and global options. Use dino8 help <command> for details on a specific command.
